# brain_classifier
Project to classify healthy and damaged brains with alzheimer or a tumor
## Classes
- 0 Healthy
- 1 Alzheimer
- 2 Tumor
## Datasets
- https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
- https://www.kaggle.com/datasets/borhanitrash/alzheimer-mri-disease-classification-dataset
- The alzheimer dataset is currently no longer available and you need to download it here:
- https://www.kaggle.com/datasets/pierpreissner/alzheimer-dataset-for-fds
- normally the dataset will be downloaded as a .zip with the name archive.zip
- It is important that the zip Files are in the google colab for the brain_classifier.ipynb to work
